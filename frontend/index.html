영수증 이미지를 올리면 자동으로 읽고, 분류하고, 정리하는 가계부 서비스

업로드한 영수증 이미지를 OCR과 AI로 분석해서 거래 내역을 자동 정리합니다. 수동 입력을 줄이고, 월별 소비 흐름과 예산 현황까지 한 화면에서 확인할 수 있습니다.

핵심 기능

이미지 업로드부터 소비 분석, 구독 결제까지 한 흐름으로 연결했습니다.

영수증 업로드 + OCR 추출

이미지를 올리면 텍스트를 자동 추출하고, 금액과 날짜 같은 핵심 정보를 빠르게 읽어옵니다.

AI 자동 분류

추출된 데이터를 기반으로 카테고리를 추천하고, 거래내역 정리를 더 빠르게 끝낼 수 있습니다.

구독/사용량 관리

무료/유료 플랜에 따라 OCR 사용량과 계정 상태를 확인하고 결제 흐름까지 연결할 수 있습니다.

사용 흐름

복잡한 입력 없이 몇 단계만 거치면 거래내역이 정리됩니다.

STEP 01

로그인

계정을 만들거나 로그인해서 개인 데이터 공간으로 진입합니다.

STEP 02

이미지 업로드

영수증 사진을 올리면 OCR이 텍스트를 추출하고 AI 분석을 준비합니다.

STEP 03

거래내역 검토

자동으로 채워진 금액, 날짜, 카테고리를 확인하고 필요한 값만 수정합니다.

STEP 04

대시보드 확인

월별 소비 추이, 최근 거래, 예산 현황을 대시보드에서 바로 확인합니다.
